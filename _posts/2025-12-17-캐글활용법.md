## 1. 튜토리얼 성격의 baseline 코드 공부하기
- competitions -> getting started -> see all (도메인/기술별 대회 확인 가능)
- code -> notebooks -> hotness 대신 most votes (성능이 좋을 확률 높음)
- 오른쪽 위 점세개 -> copy&edit notebook -> 내 머신에 카피됨 -> session 켜기 ->  주피터노트북처럼 사용가능
- 데이터 mount하기: 오른쪽 add input (캐글에서 데이터셋 불러오기), upload data(로컬에서 데이터셋 불러오기)
- 오른쪽 session options:
  accelerator - gpu 사용할것인지? (tpu가능, 구글자회사 캐글)
  pin to original env - 이전버전 환경사용 (그러나 코딩중 에러 뜰수 있음) -> 선택후 session 다시 켜기

## 2. 내 프로젝트에 활용하는 법
